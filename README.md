# Gabriel's Portfolio Website

A Lord of the Rings-inspired personal portfolio website for Gabriel, a Computer Science student.
This project uses pure HTML, CSS, and Vanilla JS, and is designed to be deployed on GitHub Pages.

## Features
- **LOTR Theme**: Light mode (Parchment) and Dark mode (Shadow)
- **Responsive Design**: Mobile-first layout with smooth transitions
- **Scroll Animations**: Dynamic reveal of elements upon scrolling
- **Contact Form**: Integration with Formspree

## File Structure
- `index.html`: Main HTML file
- `assets/css/style.css`: Styles and CSS variables
- `assets/js/main.js`: Dark mode toggle, animations, and form logic
- `assets/images/`: Images including profile picture
- `assets/docs/`: Resume PDF (placeholder)

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
